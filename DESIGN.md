# Design System - Sharehood

## צבעים (Colors)

| תפקיד | קוד צבע (Hex) | שימוש |
| :--- | :--- | :--- |
| **Primary** | `#2E7D32` | ירוק כהה - משמש לכפתורים מרכזיים (כמו "אני רוצה להשאיל"). |
| **Secondary** | `#1976D2` | כחול - משמש לקישורים, אייקונים של מיקום וניווט. |
| **Accent** | `#FFA000` | כתום - משמש להדגשות, כוכבי דירוג והתראות חשובות שצריכות למשוך תשומת לב. |
| **Background** | `#F5F5F5` | אפור בהיר - משמש כרקע הכללי של האפליקציה כדי לתת מראה נקי ומודרני. |
| **Text** | `#212121` | שחור פחם - משמש לטקסט המרכזי ולקריאות מקסימליות על הרקע הבהיר. |
| **Error / שגיאה** | `#D32F2F` | אדום - משמש להודעות שגיאה בתשלום או שדות חסרים בטפסים. |
| **Success / הצלחה** | `#388E3C` | ירוק בהיר - משמש להודעות אישור לאחר תשלום מוצלח או העלאת פריט. |

## טיפוגרפיה (Typography)

| תפקיד | שם הפונט | משקל | גודל |
| :--- | :--- | :--- | :--- |
| **Heading** | Heebo / Assistant | Bold | 24px - 32px |
| **Body** | Heebo / Assistant | Regular | 16px |
| **Caption / Label** | Heebo / Assistant | Medium | 12px - 14px |

## ריווח (Spacing)

* **יחידת בסיס (Base Unit):** 8px (כל הריווחים באפליקציה יהיו כפולות של 8, למשל: 16px, 24px, 32px)
* **Border Radius (עיגול פינות):** 8px (מעניק מראה ידידותי ומודרני לכפתורים ולכרטיסי המוצר)

## סגנון רכיבים (Components)

### כפתורים (Buttons)
* **עיצוב:** פינות מעוגלות (8px).
* **צבע:** צבע מילוי primary (ירוק `#2E7D32`), טקסט לבן בולט במרכז.
* **אינטראקציה:** במצב מעבר עכבר (hover) הצבע הופך לירוק כהה יותר.

### כרטיסים (Cards)
* **רקע:** לבן עם צל עדין (drop shadow).
* **מבנה:** פינות מעוגלות (8px), ריווח פנימי (padding) של 16px בין התמונה לטקסט.

### שדות קלט (Inputs)
* **עיצוב:** מסגרת (border) דקה בצבע אפור, פינות מעוגלות (4px-8px).
* **טקסט:** טקסט רמז (placeholder) בגוון אפור בהיר.

### ניווט (Navigation)
* **עיצוב:** סרגל תחתון קבוע עם אייקונים מינימליסטיים בשחור/אפור.
* **מצב נבחר:** הרכיב הנבחר מודגש בצבע ה-secondary (כחול `#1976D2`).
